 # Personal Banking Simulator 🏦

A simulated banking platform with authentication,
transfers, deposits, withdrawals and statements.

## Live Demo
[Click here](YOUR_STREAMLIT_URL)

## Features
- Secure signup/login with bcrypt password hashing
- Auto-generated account numbers
- Deposit, withdraw and transfer between accounts
- Real-time balance tracking with line chart
- Full transaction statement with CSV export
- Spending insights and outflow breakdown

## Tools Used
- Python, Streamlit, bcrypt, Plotly, Pandas

## How to Run Locally
pip install streamlit pandas numpy plotly bcrypt
streamlit run app.py

## Note
Educational simulation only — no real banking
or financial data involved.
